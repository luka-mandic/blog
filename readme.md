## About Blog

A blog built with the Laravel framework for the backend and Bootstrap for the frontend. 

The blog contains:

<ul>
	<li>CRUD functionality</li>
	<li>Authentication</li>
	<li>Comments and replies on posts</li>
	<li>Post likes and dislikes</li>
	<li>Post tags</li>
</ul>
